## 📦 依赖安装指南

### RapidOCR ONNX（推荐）

```bash
cd desktop

# 安装 Node.js 绑定
npm install rapidocr-onnxruntime

# 下载模型文件（放 build/ocr-models/）
# 下载地址：https://github.com/RapidAI/RapidOCR/releases
# - ch_PP-OCRv4_rec_infer.onnx (25MB)
# - ch_PP-OCRv4_det_infer.onnx (3MB)
```

### 备选：PaddleOCR Node.js

```bash
# 体积稍大，但准确率略高
npm install node-paddleocr
```

## ⚠️ 常见问题预案

### 问题 1：Electron 截图权限（Mac）
**现象：** 截图是黑屏
**解决：** 引导用户在「系统设置 → 隐私与安全性 → 屏幕录制」中授权

### 问题 2：OCR 引擎加载失败
**现象：** ONNX Runtime 初始化失败
**解决：** 降级到稳定版，或者提供备用的 WebAPI 方案

### 问题 3：识别速度慢
**排查：**
- 是否是 Debug 模式？Release 模式快 3~5 倍
- 是否启用了 INT8 量化？
- 是否裁剪了多余区域？

## 🎯 验收标准

正式发布前必须满足：

1. **速度**：区域识别 < 200ms，全屏识别 < 500ms
2. **准确率**：普通聊天场景 > 95%，LLM 修复后 > 99%
3. **稳定性**：连续识别 100 次不崩溃
4. **内存占用**：OCR 引擎常驻内存 < 150MB
5. **兼容性**：Windows 10+ / macOS 12+ 双平台正常工作
