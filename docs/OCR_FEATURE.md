# 屏幕 OCR 识别功能说明

## ✨ 功能概述

让任何聊天软件的内容都能被 AI 分析——不需要 API，不需要数据库权限，屏幕上能看到的就能识别。

---

| 特性 | 状态 | 说明 |
|------|------|------|
| **全局快捷键** | ✅ 已实现 | `Ctrl+Shift+D` 一键触发 |
| **自动截图** | ✅ 已实现 | 自动捕获当前屏幕 |
| **聊天结构化** | ✅ 已实现 | 自动识别「昵称 时间: 内容」格式 |

## ⌨️ 使用方式

### 方式一：按钮触发
1. 切换到「军师」页面
2. 点击右上角「📸 识别屏幕」按钮
3. 等待识别完成
4. 勾选需要导入的对话
5. 点击「🧠 导入军师分析」

### 方式二：全局快捷键（推荐）
任何时候按下 `Ctrl+Shift+D`（Mac 下 `Cmd+Shift+D`）：
- 自动弹出搭言窗口
- 自动开始屏幕识别
- 识别完成直接展示结果

## 🔧 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                     触发层                                │
│   全局快捷键 ←─────────────→ 手动按钮触发              │
└──────────────────────────────┬──────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────┐
│                     捕获层                                │
│   Electron desktopCapturer → 屏幕截图                   │
└──────────────────────────────┬──────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────┐
│                     预处理层                              │
│   区域裁剪 → 对比度增强 → 二值化 → 去噪               │
└──────────────────────────────┬──────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────┐
│                     OCR 引擎层                           │
│   RapidOCR (ONNX) → 文字检测 + 文字识别               │

                               ↓

│                     后处理层                              │
│   聊天格式解析 → 昵称/时间/内容分离 → 去重合并        │
│   LLM 语义修复：修正 OCR 识别错误                       │
└──────────────────────────────┬──────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────┐
│                     输出层                                │
│   结构化 JSON → 联系人匹配 → 军师模块导入 → AI 分析   │
└──────────────────────────────────────────────────────────┘
```

## ⚖️ 法律合规说明

1. **不注入任何进程** - 纯公开 API 截图，不碰其他软件内存
2. **用户主动触发** - 不后台偷偷截图，用户明确授权后才工作
3. **纯本地处理** - 所有 OCR 在本地完成，不上传任何图像数据
4. **可审计** - 截图文件仅存临时目录，识别完成立即删除

---

## 📝 已知限制

1. **暗色主题** - 部分软件深色主题可能影响识别率（需要预处理优化）
2. **表情包** - 目前只识别文字，图片内容不识别
3. **长消息换行** - 跨气泡的长消息可能被截断（后续用 LLM 合并）
4. **英文昵称** - 中文昵称识别更准确，英文次之

---

## 💡 使用技巧

1. **对齐聊天窗口** - 识别前把聊天窗口最大化，识别率更高
2. **滚动到最新消息** - 确保想分析的内容完整显示在屏幕上
3. **优先识别右边** - 对方发送的消息在右侧，识别效果更好
4. **配合剪贴板** - 对于可以复制的内容，优先用复制方式更准确
