# 🧠 聊天军师功能说明

## 📋 功能概览

聊天军师是嗒言的核心高级功能，提供深度对话分析和策略建议。

---

## 🎯 主要功能

### 1. 目标设定 🎯

**预设目标**：
- 说服客户买单
- 砍价/压价
- 提加薪/升职
- 拒绝借钱/求助
- 道歉/挽回关系
- 争论/说服对方
- 自定义目标

### 2. 局势判断 📊

**胜率评估**：
- 0-100% 胜率数字
- 进度条可视化展示
  - 🟢🟢🟢🟢🟢 绿色（70-100%）：优势明显
  - 🟡🟡🟡🟡🟡 黄色（40-69%）：势均力敌
  - 🔴🔴🔴🔴🔴 红色（0-39%）：劣势明显

**局势分析**：
- ✓ 优势：己方优势描述
- ✗ 劣势：需要注意的风险

### 3. 意图分析 🎭

**分析维度**：
- **表面意图**：对方说什么
- **真实意图**：对方真正想要的
- **情绪状态**：对方的情绪（积极/中性/消极/焦虑/愤怒等）
- **核心需求**：对方最在乎什么

### 4. 风险预警 ⚠️

识别潜在风险和陷阱，提前预警避免失误。

### 5. 策略建议 💡

**应该做的** ✅
- 3条具体行动建议
- 帮助达成目标的具体步骤

**绝对不要做的** ❌
- 3条禁忌提醒
- 避免踩坑

**推荐语气** 🎨
- 根据情境推荐最佳语气
- 友好/正式/强硬/委婉/共情

### 6. 推荐话术 💬

**3条精心设计的话术**：
1. 每条话术都有风格标签
2. 像真人聊天，自然口语化
3. 15-30字，简短有力

**操作**：
- 📋 复制：复制到剪贴板
- 🚀 发送到微信：一键复制并粘贴到微信输入框

## 💡 使用示例

### 场景：客户砍价

**输入**：
```
客户：这个价格太贵了，能便宜点吗？
```

**选择目标**：砍价/压价

**AI 分析结果**：

📊 **局势判断**（胜率 65%）
- ✓ 优势：产品质量好，有品牌背书
- ✗ 劣势：客户有竞品对比

🎭 **意图分析**
- 表面：要求降价
- 真实：确认自己没买亏，想要心理安慰
- 情绪：理性谈判，略带焦虑
- 需求：安全感

💡 **策略建议**
- ✅ 强调价值而非价格
- ✅ 可以给优惠但要有条件
- ✅ 制造稀缺感
- ❌ 不要直接拒绝
- ❌ 不要轻易让步

💬 **推荐话术**
1. 友好："您的顾虑我理解，确实价格不便宜～"
2. 正式："我们的产品质量有保证，您可以对比一下"
3. 委婉："要不这样，我给您申请个会员价？"

## 🔧 技术实现

### AI Prompt 设计

**分析维度**：
1. 局势判断（胜率、优劣势）
2. 意图分析（表面、真实、情绪、需求）
3. 风险预警
4. 策略建议（应该做、不应该做）
5. 推荐话术（3条）

**输出格式**：JSON 格式
```json
{
  "winRate": 65,
  "advantages": "你的优势",
  "disadvantages": "你的劣势",
  "surfaceIntent": "表面意图",
  "realIntent": "真实意图",
  "emotion": "情绪状态",
  "needs": "核心需求",
  "risks": "潜在风险",
  "shouldDo": ["建议1", "建议2", "建议3"],
  "shouldNotDo": ["禁忌1", "禁忌2", "禁忌3"],
  "recommendedTone": "友好",
  "replies": [
    {"style": "友好", "content": "话术1"},
    {"style": "正式", "content": "话术2"},
    {"style": "委婉", "content": "话术3"}
  ]
}
```

## 💬 常见问题

**Q1：胜率是怎么计算的？**
A：根据对话内容、对方语气、市场行情等因素综合评估。

**Q2：话术可以直接用吗？**
A：AI 生成的话术是参考模板，建议根据实际情况调整后使用。

**Q3：如何提高胜率？**
A：真诚沟通 + 了解对方需求 + 提供价值 > 套路技巧

**Q4：可以自定义目标吗？**
A：可以，选择"自定义目标"即可输入具体场景。
