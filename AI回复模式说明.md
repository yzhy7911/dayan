# 搭言 AI 回复模式说明

## 📋 核心改进

**简化回复生成逻辑**：每次只生成 1 条回复，而不是多条

---

## 🎯 两种使用模式

### 模式 1：全部风格（推荐）✨

**功能**：一次性生成 5 种不同风格的回复

**使用方法**：
1. 输入对方消息
2. 选择"✨ 全部风格"
3. 点击"生成回复"
4. 查看 5 张卡片，每张一条不同风格的回复

**示例**：
```
用户输入："你好吗？"

生成结果：

│ 挺好的呀，刚吃完饭~      │
│ [复制] [发送到微信]      │
└─────────────────────────┘
┌─────────────────────────┐
│ 💼 正式风格              │
│ 我很好，感谢关心。       │
│ [复制] [发送到微信]      │
└─────────────────────────┘
┌─────────────────────────┐
│ 😄 幽默风格              │
│ 哈哈，还活着！刚被夸了😄 │
│ [复制] [发送到微信]      │
└─────────────────────────┘
┌─────────────────────────┐
│ ⚡ 简洁风格              │
│ 很好，你呢？             │
│ [复制] [发送到微信]      │

│ 💝 共情风格              │
│ 我挺好的，你问这个是     │
│ 有什么事吗？              │
│ [复制] [发送到微信]      │
└─────────────────────────┘
```

---

### 模式 2：单一风格

**功能**：生成特定风格的 1 条回复

**使用方法**：
1. 输入对方消息
2. 选择一种风格（如"💬 友好"）
3. 点击"生成回复"
4. 查看 1 张卡片，该风格的一条回复

**示例**：
```
用户输入："老板让我加班，我不想加班"

选择风格："💬 友好"
生成结果：
┌─────────────────────────┐
│ 💬 友好风格              │
│ 好的领导，我处理完手头   │
│ 工作就过来加班～         │
│ [复制] [发送到微信]      │
└─────────────────────────┘
```

## 💡 使用技巧

### 场景 1：不知道用什么风格
→ 使用"✨ 全部风格"，一次看遍所有风格，选择最喜欢的

### 场景 2：明确知道要什么风格
→ 直接选择对应风格，快速生成

### 场景 3：想要换个风格
→ 直接点击另一种风格按钮，生成新回复

### 场景 4：对生成结果不满意
→ 再次点击"生成回复"按钮，AI 会生成不同的回复

---

## 🔄 工作流程

```
1. 用户输入消息
   ↓
2. 选择风格
   ├─ ✨ 全部风格 → 生成 5 种风格
   └─ 💬💼😄⚡💝🎯 → 生成 1 种风格
   ↓
3. AI 生成回复
   └─ 每次只生成 1 条回复（全部风格生成 5 条不同风格的）
   ↓
4. 前端展示
   └─ 多张卡片，每张 1 条回复
   ↓
5. 用户选择
   ├─ 复制到剪贴板
   ├─ 发送到微信
   └─ 重新生成
```

---

## ✨ 核心优势

1. **简单直接**：每条回复都是完整的，不需要拆分
2. **选择多样**：6 种风格可选
3. **操作直观**：多张卡片，一目了然
4. **易于决策**：看一遍就能选最喜欢的

---

## ⚠️ 注意事项

1. **全部风格**会生成 5 条回复，可能需要几秒钟
2. **单一风格**生成较快，约 1-2 秒
3. 如果生成失败，会显示错误提示，可以重试
4. 回复内容为 AI 生成，可能有偏差，请酌情使用
